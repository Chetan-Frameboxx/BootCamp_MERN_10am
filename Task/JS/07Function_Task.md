# JavaScript Function Practice Tasks

**Topics Covered:**

* Normal Functions
* Arrow Functions
* Anonymous Functions
* IIFE (Immediately Invoked Function Expressions)

---

## Normal Function Tasks

1. Create a normal function that takes two numbers and returns their sum.

2. Write a normal function that checks whether a given number is even or odd.

3. Create a normal function that takes an array of numbers and returns the largest value.

4. Write a normal function that counts how many vowels are present in a given string.

5. Create a normal function that prints numbers from 1 to N using a loop.

---

## Arrow Function Tasks

6. Create an arrow function that returns the square of a number.

7. Write an arrow function that checks if a string is a palindrome.

8. Create an arrow function that takes an array and returns the total sum of its elements.

9. Write an arrow function that returns the length of the longest word in a sentence.

10. Create an arrow function that filters out negative numbers from an array.

---

## Anonymous Function Tasks

11. Write an anonymous function assigned to a variable that converts Celsius to Fahrenheit.

12. Create an anonymous function that checks whether a number is prime.

13. Write an anonymous function that reverses a given string.

14. Create an anonymous function that finds the factorial of a number using a loop.

15. Write an anonymous function that counts how many times a specific value appears in an array.

---

## IIFE Tasks

16. Write an IIFE that prints "JavaScript Functions Started" to the console.

17. Create an IIFE that calculates and logs the sum of numbers from 1 to 10.

18. Write an IIFE that checks whether a hardcoded number is positive, negative, or zero.

19. Create an IIFE that reverses a hardcoded string and logs the result.

20. Write an IIFE that creates an array, loops through it, and prints only even numbers.
