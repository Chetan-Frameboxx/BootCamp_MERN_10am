## Array Tasks (15)

1. Create an array of 5 colors. Print the first and last color.
2. Create an array of 4 numbers. Replace the second number with a new value.
3. Create an array of 3 fruits. Add one fruit at the end using `push()`.
4. Create an array of 3 items. Remove the last item using `pop()`.
5. Create an array of countries. Add one country at the beginning using `unshift()`.
6. Remove the first element of an array using `shift()`.
7. Create an array and find its length.
8. Create an array of names and check if a specific name exists using `includes()`.
9. Join an array of 3 words into one string using `join()`.
10. Convert a string into an array using `split()`.
11. Create an array and copy a part of it using `slice()`.
12. Replace items in an array using `splice()` (without looping).
13. Merge two arrays using `concat()`.
14. Sort an array of strings alphabetically using `sort()`.
15. Reverse an array using `reverse()`.
