# JavaScript Interview Q&A –  Introduction to JS

## 1. What is JavaScript?
**Answer:** JavaScript is a programming language used to make web pages interactive. It runs in web browsers and can also run on servers using Node.js.

## 2. Who developed JavaScript?
**Answer:** JavaScript was developed by Brendan Eich in 1995 while working at Netscape.

## 3. What is the difference between Java and JavaScript?
**Answer:**  
- Java → compiled, runs on JVM, used for apps, strongly typed.  
- JavaScript → interpreted, runs in browsers, loosely typed, used for web interactivity.

## 4. Where can JavaScript be used?
**Answer:**  
- Client-side (browser) web development  
- Server-side (Node.js) development  
- Mobile apps  
- Desktop apps

## 5. How can you run JavaScript code in a browser?
**Answer:**  
- Using `<script>` tags in HTML  
- Through the browser console

## 6. What are the ways to include JavaScript in a webpage?
**Answer:**  
1. Inline: `<button onclick="alert('Hi')">Click</button>`  
2. Internal: Inside `<script>` tags in HTML  
3. External: Using a `.js` file with `<script src="file.js"></script>`

## 7. What is the difference between client-side and server-side JavaScript?
**Answer:**  
- Client-side → runs in browser, interacts with the user interface  
- Server-side → runs on server, handles database and business logic

## 8. How do you write your first JavaScript code?
**Answer:**  
```js
console.log("Hello World");

`````

## 9. What is the purpose of `console.log()`?
**Answer:** To print messages or values in the browser console for debugging or testing.

## 10. What is a JavaScript engine?
**Answer:** A program that executes JavaScript code. Example: V8 engine in Google Chrome.

## 11. Can JavaScript be used for styling web pages?
**Answer:** No, CSS is used for styling. JavaScript can modify styles dynamically but cannot replace CSS.

## 12. What is ECMAScript?
**Answer:** ECMAScript is the standard specification on which JavaScript is based.

## 13. Which browsers support JavaScript?
**Answer:** All modern browsers like Chrome, Firefox, Edge, Safari, and Opera.

## 14. Is JavaScript case-sensitive?
**Answer:** Yes, identifiers, keywords, and variable names are case-sensitive.

## 15. How do you write comments in JavaScript?
**Answer:**  
- Single-line: `// This is a comment`  
- Multi-line: `/* This is a comment */`

## 16. What are the basic building blocks of JavaScript in the first lecture?
**Answer:** Statements, expressions, comments, and basic output using `console.log()`.

## 17. Can JavaScript change HTML content?
**Answer:** Yes, using DOM manipulation, but this is taught in later lectures.

## 18. What is the syntax for a basic JavaScript statement?
**Answer:**  
```js
console.log("message");
````
