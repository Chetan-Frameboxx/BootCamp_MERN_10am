# JavaScript Loop Practice Tasks 

## 1–5: Basic `for` loop tasks
1. Print numbers from 1 to 10.  
2. Print even numbers between 1 and 20.  
3. Print odd numbers between 1 and 20.  
4. Print numbers from 10 to 1 in reverse order.  
5. Calculate the sum of numbers from 1 to 50.  

## 6–10: `while` loop tasks
6. Print numbers from 1 to 15 using `while`.  
7. Print all multiples of 3 between 1 and 30 using `while`.  
8. Calculate the factorial of a number using `while`.  
9. Print Fibonacci series up to 50 using `while`.  
10. Print numbers from 50 down to 1 using `while`.  

## 11–15: `do...while` loop tasks
11. Print numbers from 1 to 10 using `do...while`.  
12. Print even numbers between 2 and 20 using `do...while`.  
13. Calculate the sum of numbers from 1 to 20 using `do...while`.  
14. Print all odd numbers from 1 to 15 using `do...while`.  
15. Print numbers from 10 down to 1 using `do...while`.  

## 16–20: `for...of` loop tasks
16. Iterate over an array of numbers `[1, 2, 3, 4, 5]` and print each number.  
17. Iterate over an array of names `["Alice", "Bob", "Charlie"]` and print a greeting.  
18. Calculate the sum of numbers in an array `[5, 10, 15, 20]`.  
19. Print only the even numbers from an array `[3, 4, 7, 10, 12]`.  
20. Concatenate all strings in an array `["Hello", " ", "World"]`.  

## 21–24: `for...in` loop tasks
21. Iterate over an object `{name: "Alice", age: 25, city: "Delhi"}` and print keys and values.  
22. Sum all numeric values of an object `{a: 10, b: 20, c: 30}`.  
23. Print all keys of an object `{x: 1, y: 2, z: 3}`.  
24. Print all values of an object `{fruit: "Apple", color: "Red"}`.  

## 25–27: `forEach` loop tasks
25. Use `forEach` to print all elements of `[1, 2, 3, 4, 5]`.  
26. Use `forEach` to calculate the sum of `[10, 20, 30]`.  
27. Use `forEach` to print each name in `["John", "Jane", "Jim"]` with its index.
